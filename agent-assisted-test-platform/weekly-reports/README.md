# Weekly Reports

本目录用于保存 Agent 辅助测试平台实习项目的每周进展反馈。

## 命名规范

```text
week-01.md
week-02.md
week-03.md
...
```

## 内容要求

每周至少包含：

- 本周目标与完成情况。
- 可运行、可检查的产出。
- 测试和验证结果。
- 重要设计决策及其状态。
- Agent 参与点与人工验证方式。
- Agent 的错误建议或需要人工修正的内容。
- 风险、阻塞和需要导师支持的事项。
- 下周计划。

建议基于 `template.md` 复制生成新周报。周报不能只写“完成百分比”，必须链接到代码、文档、测试报告或演示记录。
